# 프로젝트 요약

## 개요

사업자등록증 정보 추출기는 사업자등록증 이미지, PDF, ZIP 파일을 업로드해 거래처 등록에 필요한 정보를 자동 추출하는 로컬 웹 애플리케이션입니다.

## 주요 기능

- 여러 파일 및 ZIP 파일 일괄 업로드
- PDF 텍스트 추출 및 이미지 렌더링 기반 AI OCR
- Ubion LiteLLM 연동 비전 모델 사용
- JSON 후처리와 품질 검사를 통한 결과 정규화
- 카카오 주소 검색 API를 이용한 우편번호 보정
- 사내 거래처 등록 양식에 맞춘 엑셀 다운로드
- 화면 푸터에 현재 사용 중인 AI 모델명 표시

## 현재 기본 모델

```env
UBION_VISION_MODEL=gpt-4o
```

## 핵심 파일

- `server.ts`: Express 서버, 파일 업로드, AI OCR, 카카오 우편번호 보정
- `src/App.tsx`: React 화면, 파일 큐, 결과 그리드, 엑셀 다운로드
- `.env.example`: 로컬 실행에 필요한 환경 변수 예시
- `run_local.bat`: Windows 로컬 실행 스크립트

## 보안 주의

`.env`에는 실제 API 키가 들어가므로 GitHub에 올리지 않습니다. 공개 저장소에는 `.env.example`만 포함합니다.
